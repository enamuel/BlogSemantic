Projecte: Blog amb HTML Semàntic

Publicat per: Emanuel el 05 de novembre de 2025

En aquest projecte he decidit allotjar el blog en GitHub, per familiaritzar-me amb el lloc web i les seves característiques per "desplegar" llocs web a l'internet amb ajuda de GitHub Actions.
