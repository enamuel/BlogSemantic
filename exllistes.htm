Tots els Tipus de Llistes HTML

1. Llista Desordenada ("ul")

Utilitzada per a elements sense un ordre particular. Cada element es marca amb una vinyeta.

  • Element 1
  • Element 2
  • Element 3

2. Llista Ordenada ("ol")

Utilitzada per a elements que tenen un ordre específic. Es numeren automàticament.

  1. Primer pas
  2. Segon pas
  3. Tercer pas
---

3. Llista Desordenada amb Enllaços e Imatges

Una combinació habitual per a galeries o menús.

  • Lleó
  • Elefant
---

4. Llista de Descripció ("dl")

Ideal per a termes i les seves definicions.

HTML
Llenguatge d'etiquetes d'hipertext, utilitzat per estructurar el contingut d'una pàgina web.
CSS
Fulls d'estil en cascada, utilitzats per estilitzar i dissenyar l'aparença d'una pàgina web.
---

5. Llista Anidada (Combinant "ul" i "oi")

Per crear estructures jeràrquiques amb subllistes.

  • Aliments
    • Fruites
      1. Poma
      2. Plàtan
    • Verdures
  • Begudes
---

6. Exemple de Llista Estilitzada com a Menú

Una llista desordenada ("ul") transformada en una barra de navegació amb CSS.

  • Inici
  • Productes
  • Serveis
  • Contacte
