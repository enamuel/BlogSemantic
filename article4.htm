Introducció al CSS

Publicat per: Emanuel el 01 d'octubre de 2025

El CSS o Cascading Style Sheets és el llenguatge de fulls de estils utilitzat per presentar i estilitzar pàgines HTML o Arxius XML (Extensible Markup Language).

L'introducció al CSS ve acompanyada amb l'introducció a les etiquetes per a fer taules: <table> <th>, <tr> i <td>, respectivament. Després tenim <caption> i <thead>, <tbody>, <tfoot> i <colgroup>, <col> per a especificar.

Fent ús dels enllaços, el full d'estils es pot enllaçar amb <link rel="stylesheet" href="fulldestils.css">.
